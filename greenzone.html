About This Project

GreenZone is a recycling program that consists of a kiosk and a smart scanner to track recycling activities within a residential community, and translates them into visual displays of a virtual forest. The project was part of a graduate course in User-Centered Design. I completed this project in collaboration with three teammates.

View design process


My role

User Experience Design + Visual Design

As the lead UX Designer of GreenZone, my contributions include developing user personas, design ideation, as well as creating, evaluating, and refining prototypes. I handled the visual design works in the project such as persona profile design, interface design, illustrations, and developed design specifications. I actively particpated in user research and usability testing sessions through interviewing, notetaking, and affinity-diagramming. Upon project completion, I designed and co-authored a website to document the project process.
