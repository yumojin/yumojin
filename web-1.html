About this project

(See a working demo of this website here!)
This website is the final project of my class Advanced Front-End Web Development. The prompt was to build a website that provides basic informations about William Shakespeare and also collects and stores subscribers' email information in a database.


Key elements

  • Designing and building the layout of the webpage without a framework system
  • Structuring the website for optimal "reading" experience, basing off of the standard Z-pattern
  • Making sure the website is responsive for mobile devices
  • Building a simple form for new email suscriptions
  • Succesfully implementing an email database using MySQL management system

Production

  • HTML5
  • CSS3
  • JavaScript
  • PHP
  • SQL
